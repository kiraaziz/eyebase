# Eyebase API: Deleting Documents

## Overview
Remove a document from your collection by its unique ID.

## Authentication
All requests require an API key with delete permission passed in the `Authorization` header.

## Deleting a Document
To permanently delete a specific document from a collection:

```javascript
// JavaScript
fetch('https://eyebase.vercel.app/go/v0.1/abc123?collectionName=users', {
  method: 'DELETE',
  headers: {
    'Authorization': 'your-api-key'
  }
})
.then(response => response.json())
.then(result => console.log(result));
```

```python
# Python
import requests

response = requests.delete(
  'https://eyebase.vercel.app/go/v0.1/abc123?collectionName=users',
  headers={
    'Authorization': 'your-api-key'
  }
)
result = response.json()
print(result)
```

Replace `abc123` with your actual document ID.

## Response Format
```json
{
  "status": {
    "message": "Document deleted successfully",
    "success": true,
    "value": 200
  },
  "data": {
    "eyeId_": "abc123"
  },
  "meta": {
    "collectionName": "users"
  }
}
```

## Error Responses

### Document Not Found
```json
{
  "status": {
    "message": "Document not found",
    "success": false,
    "value": 404
  },
  "data": null,
  "meta": null
}
```

### Permission Error
```json
{
  "status": {
    "message": "API key invalid or no permission to delete from the collection",
    "success": false,
    "value": 403
  },
  "data": null,
  "meta": null
}
```

Documents are permanently deleted and cannot be recovered, so use this endpoint with caution!
